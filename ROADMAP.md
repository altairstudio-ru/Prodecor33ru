## 📝 Заметки

- **Базовый путь (`base`)** в `vite.config.ts` отличается для Vercel (`/`) и GitHub Pages (`/Prodecor333ru/`). При изменении конфига проверять оба хостинга.
- **Логотип**: `viewBox='0 195 1299 300'` — при правке проверять Header и Footer.
- **Контакты** захардкожены в компонентах. При смене телефона/соцсетей править в `Header.tsx`, `Footer.tsx`, `ContactForm.tsx`, `Trust.tsx`.
- **UI-библиотека** (`src/app/components/ui/`) почти не используется — оставлена для будущих страниц.
